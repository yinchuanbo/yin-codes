UDP，提供更快速的连接建立和重传机制。</p>\n</li>\n</ul>\n<h3>1.4. 代码拆分和懒加载</h3>\n<ul>\n<li><p>按需加载：使用 Webpack CSS 和 JavaScript 文件。</p>\n<p>预获取：使用 <code>&lt;link rel=&quot;prefetch&quot;&gt;</code> 标记预获取非关键资源，如未来的页面。</p>\n<h3>1.6. 减少 DNS 查询</h3>\n<ul>\n<li><p>减少域名数量：减少域名的数量可以减少 DNS 查询的时间。</p>\n</li>\n<li><p>使用
